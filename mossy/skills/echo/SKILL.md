# Echo

## When To Use This Skill

Use this skill when the user asks to repeat, echo, acknowledge, or produce a small direct text response.

## Instructions

Answer directly and succinctly. If you are running inside a queued task and structured task state is available, call `record_task_result` with the original goal and the answer you produced.
